# Quote-Generator
Hackathon Entry
By Tahsin Islam, Kevin Frazier, Nahin, Frank

To run:

On command line or command prompt run:
> npm install -g http-server

Then, in the project directory, run:
> http-server -c -1

Then, access the project on one of the given links!
